---
title: "Retrieve Label Batch"
api: "GET https://api.evership.app/v1/shipments/labels/{batch_id}"
description: "Retrieve a previously created label batch"
---

Retrieves the status and labels for a previously created batch. Use the batch ID returned from the [Create Label Batch](/api-reference/labels/create-batch) endpoint.

## Path Parameters

<ParamField path="batch_id" type="string" required>
  The batch ID returned when the batch was created (e.g. `batch_pywoz0`).
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.evership.app/v1/shipments/labels/BATCH_ID \
  -H "apiKey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.evership.app/v1/shipments/labels/BATCH_ID', {
  method: 'GET',
  headers: {
    'apiKey': 'YOUR_API_KEY'
  }
});
const data = await response.json();
```

```typescript TypeScript
interface RetrieveLabelBatchResponse {
  label_format: "10_10" | "10_15" | "a4_page";
  label_batch_id: string;
  label_url: string;
}

const batchId: string = 'BATCH_ID';

const response = await fetch(
  `https://api.evership.app/v1/shipments/labels/${batchId}`,
  {
    method: 'GET',
    headers: {
      'apiKey': 'YOUR_API_KEY'
    }
  }
);
const data: RetrieveLabelBatchResponse = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.evership.app/v1/shipments/labels/BATCH_ID',
    headers={'apiKey': 'YOUR_API_KEY'}
)
data = response.json()
```
