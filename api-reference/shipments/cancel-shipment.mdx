---
title: "Cancel Shipment"
api: "POST https://api.evership.app/v1/shipments/cancel/{shipment_id}"
description: "Cancel a purchased shipment"
---

Cancels a shipment that has already been purchased. This will attempt to cancel the collection with the carrier.

## Path Parameters

<ParamField path="shipment_id" type="string" required>
  The ID of the shipment to cancel.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.evership.app/v1/shipments/cancel/SHIPMENT_ID \
  -H "apiKey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.evership.app/v1/shipments/cancel/SHIPMENT_ID', {
  method: 'POST',
  headers: {
    'apiKey': 'YOUR_API_KEY'
  }
});
const data = await response.json();
```

```typescript TypeScript
interface CancelShipmentResponse {
  status: "success" | "error";
  shipment_id: string;
  message: string;
}

const shipmentId: string = 'SHIPMENT_ID';

const response = await fetch(
  `https://api.evership.app/v1/shipments/cancel/${shipmentId}`,
  {
    method: 'POST',
    headers: {
      'apiKey': 'YOUR_API_KEY'
    }
  }
);
const data: CancelShipmentResponse = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.evership.app/v1/shipments/cancel/SHIPMENT_ID',
    headers={'apiKey': 'YOUR_API_KEY'}
)
data = response.json()
```
