---
title: "Delete Shipment"
api: "DELETE https://api.evership.app/v1/shipments"
description: "Delete one or more unpurchased shipments"
---

Deletes one or more shipments. Supports batch deletion by passing multiple comma-separated shipment IDs. Only unpurchased shipments can be deleted.

## Query Parameters

<ParamField query="shipment_ids" type="string" required>
  Comma-separated list of shipment IDs to delete (e.g. `"id1,id2,id3"`).
</ParamField>

<RequestExample>
```bash Single delete
curl -X DELETE "https://api.evership.app/v1/shipments?shipment_ids=SHIPMENT_ID" \
  -H "apiKey: YOUR_API_KEY"
```

```bash Batch delete
curl -X DELETE "https://api.evership.app/v1/shipments?shipment_ids=ID_1,ID_2,ID_3" \
  -H "apiKey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.evership.app/v1/shipments?shipment_ids=SHIPMENT_ID', {
  method: 'DELETE',
  headers: {
    'apiKey': 'YOUR_API_KEY'
  }
});
const data = await response.json();
```

```typescript TypeScript
interface DeleteShipmentResult {
  shipment_id: string;
  status: "deleted" | "not_found" | "error";
  error_message?: string;
}

interface DeleteShipmentsResponse {
  summary: {
    deleted: number;
    failed: number;
    total: number;
  };
  results: DeleteShipmentResult[];
}

const shipmentIds: string[] = ['ID_1', 'ID_2', 'ID_3'];

const response = await fetch(
  `https://api.evership.app/v1/shipments?shipment_ids=${shipmentIds.join(',')}`,
  {
    method: 'DELETE',
    headers: {
      'apiKey': 'YOUR_API_KEY'
    }
  }
);
const data: DeleteShipmentsResponse = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.evership.app/v1/shipments',
    headers={'apiKey': 'YOUR_API_KEY'},
    params={'shipment_ids': 'SHIPMENT_ID'}
)
data = response.json()
```
