---
title: "Purchase Shipment"
api: "POST https://api.evership.app/v1/shipments/purchase"
description: "Purchase a label for an existing shipment"
---

Purchases a shipping label for a previously created shipment. This assigns a carrier, generates a waybill, and creates a label.

## Request Body

<ParamField body="shipment_id" type="string" required>
  The ID of the shipment to purchase.
</ParamField>

<ParamField body="carrier_service" type="string">
  The carrier service to use (e.g. "dpd-express"). Leave empty to use the default or previously selected service.
</ParamField>

<ParamField body="collection_date" type="string" required>
  Requested collection date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="collection_window" type="string">
  Collection time window. Example: `anytime`.
</ParamField>

<ParamField body="label_format" type="array">
  Array of label formats to generate. Possible values: `10_10`, `10_15`, `a4_page`.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.evership.app/v1/shipments/purchase \
  -H "apiKey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "shipment_id": "SHIPMENT_ID",
    "carrier_service": "",
    "collection_date": "2025-09-17",
    "collection_window": "anytime",
    "label_format": ["10_10"]
  }'
```
</RequestExample>
