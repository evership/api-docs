---
title: "Validate Address"
api: "POST https://api.evership.app/v1/validate-address"
description: "Validate a South African address before creating a shipment"
---

Validates an address to ensure it can be used for shipping. Use this before creating shipments to catch address issues early.

## Request Body

<ParamField body="address_1" type="string" required>
  Primary street address (e.g. "14 Highland Close")
</ParamField>

<ParamField body="address_2" type="string">
  Secondary address line (e.g. apartment, unit number)
</ParamField>

<ParamField body="suburb" type="string" required>
  Suburb name
</ParamField>

<ParamField body="city" type="string" required>
  City name
</ParamField>

<ParamField body="province" type="string" required>
  Province name (e.g. "Western Cape", "Gauteng")
</ParamField>

<ParamField body="postal_code" type="string" required>
  Postal code
</ParamField>

<ParamField body="country" type="string" required>
  Country name (e.g. "South Africa")
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.evership.app/v1/validate-address \
  -H "apiKey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "address_1": "14 Highland Close",
    "address_2": "",
    "suburb": "Vredehoek",
    "city": "Cape Town",
    "province": "Western Cape",
    "postal_code": "8001",
    "country": "South Africa"
  }'
```
</RequestExample>
