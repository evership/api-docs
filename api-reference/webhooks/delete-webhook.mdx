---
title: "Delete Webhook"
api: "DELETE https://api.evership.app/v1/webhooks"
description: "Remove a webhook subscription"
---

Deletes an existing webhook subscription. You will no longer receive notifications at the webhook URL.

<RequestExample>
```bash cURL
curl -X DELETE https://api.evership.app/v1/webhooks/WEBHOOK_ID \
  -H "apiKey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.evership.app/v1/webhooks/WEBHOOK_ID', {
  method: 'DELETE',
  headers: {
    'apiKey': 'YOUR_API_KEY'
  }
});
const data = await response.json();
```

```typescript TypeScript
const webhookId: string = 'WEBHOOK_ID';

const response = await fetch(
  `https://api.evership.app/v1/webhooks/${webhookId}`,
  {
    method: 'DELETE',
    headers: {
      'apiKey': 'YOUR_API_KEY'
    }
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.evership.app/v1/webhooks/WEBHOOK_ID',
    headers={'apiKey': 'YOUR_API_KEY'}
)
data = response.json()
```
