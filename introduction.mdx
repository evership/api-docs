## Welcome to Evership

Evership provides a unified shipping API that connects you to multiple carrier services across South Africa. With a single integration, you can get rates, create shipments, print labels, and track deliveries across all supported carriers.

## Base URL

All API requests are made to:

```
https://api.evership.app/v1
```

## Getting Started

1. **Sign up** for an Evership account at [evership.co.za](https://evership.co.za)
2. **Get your API key** from the dashboard
3. **Authenticate** your requests by including your API key in the `apiKey` header
4. **Start shipping** by validating addresses, getting rates, and creating shipments
